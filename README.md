# recipe-finder-maps

1) Install Node dependencies using npm install

2) Run in development mode using npm run dev

3) Run in production mode using npm run build && npm run start

4) Run tests using npm run test or npm run test:coverage
(Note: Test coverage is not 100%, as writing test cases for all conditions was quite time-consuming.)

5) Good Lighthouse scores for performance, SEO and accessibility.
