# Recipe Finder - Technical Assessment

## Objective

Create a functional recipe application that allows users to search for meals, filter results by various criteria, and view detailed recipe information including ingredients and cooking instructions.

## Tech Stack

- Framework: Next.js 15.5.6 (Pages Router)
- Language: TypeScript 5
- Styling: TailwindCSS 4
- Testing: Jest + React Testing Library
- API: TheMealDB (free, no API key required)

## Getting Started

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.

### Run Tests

```bash
npm test              # Run all tests
npm run test:watch    # Run tests in watch mode
npm run test:coverage # Run tests with coverage report
```

### Build for Production

```bash
npm run build
npm start
```

## Submission Guidelines

When you're finished:

1. Ensure all tests pass: `npm test`
2. Ensure the build completes: `npm run build`
